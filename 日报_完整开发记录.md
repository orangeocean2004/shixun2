#### 语义边界集成策略

```
分段决策优先级（从高到低）：
1. 标题边界（强边界，不可跨越）
2. 特殊块保护（表格/代码/公式，原子保留）
3. 最大长度限制（超过max_chars时在最近句子边界拆分）
4. 语义边界（相似度<阈值时作为弱边界，优先切分）
5. 目标长度区间（尽量在target_chars附近切分）
6. 最小长度约束（小于min_chars时尝试合并相邻chunk）
```

#### QA合成功能（进阶）

每chunk生成1–3条问答对，自动校验：
- **可答性**（answerability）：答案是否可以从原文chunk中直接得出
- **忠实度**（faithfulness）：答案是否完全基于原文，不引入外部知识
- **质量评分**（quality_score）：综合可答性和忠实度的加权分

输出JSONL格式，每行包含：question, answer, source_chunk_id, title_path, answerable, faithful, quality_score
